# Content Hub Troubleshooting

**Audience:** Content team, approvers, admins  
**Owner:** TBD  
**Last reviewed:** 2026-03-30  
**Related guides:** `quick-start.md`, `roles/admin.md`

## I Cannot See Calendar, Ideas, Requests, Or Admin

These areas depend on your feature access or admin status. Ask an admin to confirm which features are enabled for your account.

## I Cannot Add Content

The `Add Content` action depends on calendar access. If it is missing or disabled, check your permissions.

## An Item Is Stuck In The Workflow

Check whether the item:

- is still in `Draft`
- is waiting in `Ready for Review`
- has blockers visible in the approvals flow

## I Cannot Find Reporting

Use the sidebar:

- `Reporting` for reporting workflows
- `Insights` for analytics and engagement views

## I Pressed A Shortcut And Nothing Happened

Some shortcuts depend on the current view or available features. Start with `?` to confirm the available shortcut list.
