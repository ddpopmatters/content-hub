## When To Use This

Use this workflow when you need to manage a strategic moment as a structured planning object rather than a single content entry.

Content peaks are useful for:

- major campaign moments
- key external dates
- time-bounded advocacy pushes
- moments where several content assets need coordination

## Before You Start

Have the core planning information ready:

- title
- start date
- end date
- priority
- owner

Optional but useful:

- campaign
- content pillar
- response mode
- required platforms
- required asset types
- linked entries
- description and notes

## Steps

1. Open `Content`.
2. Switch to `Peaks`.
3. Select `Create content peak` if you are creating a new planning object.
4. Enter the title and date window.
5. Set the priority tier and owner.
6. Add campaign, content pillar, or response mode if they help classify the peak.
7. Add required platforms and asset types if the moment needs specific outputs.
8. Link any existing entries already associated with the peak.
9. Save the peak.
10. Review the peak card or list entry to confirm the planning object now exists.

## What Happens Next

You can use the peak as the anchor for multiple entries and create content from it when execution work begins.

## Common Problems

### The peak exists but no content has been scheduled yet

That is normal. A content peak is a planning object first, not automatically a content entry.

### You are tracking a recurring format, not a one-off moment

Use `Series` instead of `Peaks` if the work is a repeating format with episodes or instalments.
