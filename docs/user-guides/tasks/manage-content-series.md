## When To Use This

Use this workflow when you need to manage a recurring format or repeatable strand of content.

Content series are useful for:

- repeatable explainer formats
- campaign strands with multiple instalments
- recurring myth-busting or data-led formats
- work that needs episode targets and review checkpoints

## Before You Start

Have the core series information ready:

- title
- owner
- current status

Optional but useful:

- target platforms
- target episode count
- review checkpoint
- campaign
- content pillar
- response mode
- linked episodes
- description and notes

## Steps

1. Open `Content`.
2. Switch to `Series`.
3. Select `Create series` if you are creating a new recurring format.
4. Enter the series title.
5. Set the owner and status.
6. Add target episode count and review checkpoint if you want formal pacing.
7. Add target platforms and any campaign or pillar tags.
8. Link existing episodes if the series has already started.
9. Save the series.

## What Happens Next

The series becomes a planning object that helps you track recurring work. You can then create the next entry from the series when the next instalment is ready.

## Common Problems

### The work is a short-lived campaign moment, not a recurring format

Use `Peaks` instead of `Series` if the work is tied to a specific date window rather than a repeated format.

### The series is no longer active

Update the status rather than leaving it looking current if the format is paused or completed.
