# Content Hub Admin Role

**Audience:** Content Hub admins  
**Owner:** TBD  
**Last reviewed:** 2026-03-30  
**Related guides:** `../overview.md`, `../tasks/run-reporting-workflows.md`

## What This Role Usually Does

- access all core navigation areas
- manage admin-only settings and user-related controls
- review the overall state of the content operation
- help resolve permission, data, or workflow issues

## Main Responsibilities

- keep access aligned to team needs
- maintain operational consistency across planning, approvals, and reporting
- help diagnose why users cannot see certain features or data

## Best Practice

- avoid making silent changes to workflow settings without warning affected users
- confirm whether a problem is a permission issue, a data issue, or a workflow issue before changing anything
