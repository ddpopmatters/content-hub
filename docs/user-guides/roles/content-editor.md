# Content Editor Role

**Audience:** Content editors and planners  
**Owner:** TBD  
**Last reviewed:** 2026-03-30  
**Related guides:** `../overview.md`, `../tasks/plan-content-in-calendar.md`, `../tasks/move-work-through-kanban.md`

## What This Role Usually Does

- create content entries
- maintain the content calendar
- move work through the standard production states
- manage ideas and requests where enabled
- prepare work for review and approval

## Main Responsibilities

- keep entry details accurate
- use the right workflow status
- hand items to approvers at the right time
- avoid leaving work stuck between stages

## Common Work Areas

- `Content`
- `Dashboard`
- `Reporting` where needed

## Handoffs

- when an item is ready, move it into review and alert the approver using the agreed internal process
- if a request belongs in a different PM tool, move the workflow there rather than forcing it into Content Hub
