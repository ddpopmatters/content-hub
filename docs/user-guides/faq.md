## What is the main workflow status model?

The streamlined workflow is `Draft`, `Ready for Review`, `Approved`, and `Published`.

## What is the difference between Reporting and Insights?

Use `Reporting` for reporting workflows and `Insights` for analytics and engagement views.

## Why can another user see tabs that I cannot?

Some content areas are feature-dependent and not visible to every user.

## Where should landing page request work live?

Use Pages Hub for the landing page request lifecycle.

## What is the fastest way to add something new?

Use `Add Content` if your account has calendar access.
