## What Content Hub Is For

Content Hub is the PM content operations workspace for planning, producing, approving, and reporting on content.

It supports day-to-day workflows such as:

- planning content on the calendar
- moving work through production states
- reviewing items in the approvals queue
- capturing ideas and content requests
- running reporting workflows

## Main Navigation Areas

The main sidebar includes:

- `Dashboard`
- `Content`
- `Insights`
- `Reporting`
- `Influencers` where enabled
- `Admin` for admins

The `Add Content` button is the fastest way to create a new content entry when calendar access is enabled.

## Main Content Areas

Inside the content workspace, users may see:

- `Calendar`
- `Peaks`
- `Series`
- `Responses`
- `Ideas`
- `Requests`

Which tabs appear depends on feature access.

## Workflow States

The streamlined production statuses are:

- `Draft`
- `Ready for Review`
- `Approved`
- `Published`

## Who Should Use It

- content editors and planners
- approvers
- admins overseeing workflow and configuration

## What It Is Not For

Content Hub is not the main landing page request tool, project workload manager, or intelligence platform. Those workflows belong in other PM tools.
